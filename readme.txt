== Description ==
Adds a Widget so you can easily embed a FTSE 100 chart anywhere 
on your Wordpress site.
== Installation ==
1. Upload the FTSE100 folder to the '/wp-content/plugins/' folder
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the widget to a panel on your site via the Wordpress Widgets menu
== Frequently Asked Questions ==

= Where does the chart come from? =
Yahoo! Finance is polled at intervals for the current chart
= How often is the chart updated? =
At least once per hour, more often if the server is not overloaded
= Why not embed the Yahoo! Finance directly = 
You can of course do this if you prefer. You might choose to use this widget for 
greater control and to take advantage of chart caching.
